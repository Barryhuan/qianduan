# Vue2版本的项目结构

```
── build  —————————————————————————————  webpack配件文件夹（一般不用修改）
│   │
│   ├── build.js  ——————————————————————  构建生产环境（打包配置）
│   │
│   ├── check-versions.js  —————————————  依赖模块版本适用的检查文件（npm，node） 
│   │
│   ├── dev-client.js  —————————————————  客服端自动刷新（热重载）
│   │
│   ├── dev-server.js  —————————————————  构建本服务器（运行npm run dev运行的是这个文件，利用构建工具构建本地服务器，检测版本，加载配置，创建服务器）
│   │
│   ├── utils.js  ——————————————————————  构建工具
│   │
│   ├── vue-loader.conf.js  ————————————  css加载相关配置
│   │
│   ├── webpack.base.conf.js  ——————————  webpack基本配置
│   │
│   ├── webpack.dev.conf.js  ———————————  webpack开发配置
│   │
│   └── webpack.prod.conf.js  ——————————  webpack生产配置
│
│
── config  ————————————————————————————  Vue基本配置文件夹（可以修改端口打包路径）
│   │
│   ├── dev.env.js  ————————————————————  开发配置
│   │
│   ├── index.js  ——————————————————————  基本配置
│   │
│   └── prod.env.js  ———————————————————  生产配置
│
│
—— node_modules  ——————————————————————  项目依赖包
│
│
── src  ———————————————————————————————  核心文件夹（主要文件存放处）
│
│   ├── assets  ————————————————————————  静态文件夹（一般用来存放css文件，css，less，sass，以及一部分外部js）
│   │   └── logo.png
│   ├── components  ————————————————————  公开组件
│   │   └── Hello.vue
│   │
│   ├── router  ————————————————————————  路由
│   │
│   ├── App.vue  ———————————————————————  根组件
│   │
│   └── main.js  ———————————————————————  入口文件
│
│
── static  ————————————————————————————  静态文件夹（主要用来存放图片）
│   │
│   └── .gitkepp  ——————————————————————  git用来控制版本的文件
│
│
│—— .babelrc  ——————————————————————————  JS语法ES6转ES5的配置文件
│
│—— .editorconfig  —————————————————————  编辑器配置（用来控制代码格式的文件）
│   
│—— .eslintignore ————————————————————— 代码规范检查的忽略配置文件
│
│—— .eslintrc  —————————————————————  代码规范检查的配置文件
│
│—— .gitgnore  —————————————————————————  git忽略配置文件
│
│—— .postcssrc.js  —————————————————————  项目进行CSS转换时会用到的配置文件
│
├── index.html  ————————————————————————  主页面文件
│
├── package.json  ——————————————————————  项目信息文件
│
└── README.md  —————————————————————————  项目介绍说明的文件
```
